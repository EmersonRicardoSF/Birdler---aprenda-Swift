Controle de Fluxo

O controle de fluxo em programação é usado para determinar o caminho que um programa deve seguir durante sua execução. Em Swift, isso é feito usando condicionais e loops.

Condicionais: if, else if, else

As condicionais permitem que você execute diferentes blocos de código com base em certas condições.

Estrutura do if

Descrição: Avalia uma condição e executa o bloco de código associado se a condição for verdadeira.

let idade = 18

if idade >= 18 {
    print("Você é maior de idade.")
}

Aqui, o bloco de código dentro do if será executado porque a condição idade >= 18 é verdadeira.

Estrutura do else

Descrição: Executa um bloco de código se todas as condições anteriores (if e else if) forem falsas.

let idade = 16

if idade >= 18 {
    print("Você é maior de idade.")
} else {
    print("Você é menor de idade.")
}

Neste caso, como idade < 18, o bloco de código dentro do else será executado.

Estrutura do else if

Descrição: Avalia uma segunda condição se a condição anterior (if) for falsa.

let pontuacao = 85

if pontuacao >= 90 {
    print("Nota A")
} else if pontuacao >= 80 {
    print("Nota B")
} else if pontuacao >= 70 {
    print("Nota C")
} else {
    print("Nota D")
}

Aqui, a condição pontuacao >= 80 é verdadeira, então o bloco de código associado ao else if será executado.
