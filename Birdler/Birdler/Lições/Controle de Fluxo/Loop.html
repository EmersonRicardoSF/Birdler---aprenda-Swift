Loops

Os loops são usados para executar repetidamente um bloco de código até que uma condição específica seja satisfeita. Em Swift, existem três tipos principais de loops: for-in, while e repeat-while.

For-in Loop

O loop for-in é usado para iterar sobre sequências como intervalos de números, arrays, dicionários, strings e outras coleções.

Iterando sobre um Intervalo de Números

Descrição: Itera sobre um intervalo de números especificado.

for i in 1...5 {
    print(i)
}
// Saída:
// 1
// 2
// 3
// 4
// 5

Iterando sobre um Array

Descrição: Itera sobre cada elemento em um array.

let frutas = ["Maçã", "Banana", "Laranja"]

for fruta in frutas {
    print(fruta)
}
// Saída:
// Maçã
// Banana
// Laranja

Iterando sobre um Dicionário

Descrição: Itera sobre cada par chave-valor em um dicionário.

let capitais = ["Brasil": "Brasília", "Japão": "Tóquio"]

for (pais, capital) in capitais {
    print("A capital do \(pais) é \(capital).")
}
// Saída:
// A capital do Brasil é Brasília.
// A capital do Japão é Tóquio.

Iterando sobre uma String

Descrição: Itera sobre cada caractere em uma string.

let palavra = "Swift"

for letra in palavra {
    print(letra)
}
// Saída:
// S
// w
// i
// f
// t

Usando Índices

Descrição: Itera sobre índices de uma coleção.

let numeros = [10, 20, 30, 40, 50]

for (indice, valor) in numeros.enumerated() {
    print("Índice \(indice): \(valor)")
}
// Saída:
// Índice 0: 10
// Índice 1: 20
// Índice 2: 30
// Índice 3: 40
// Índice 4: 50

While Loop

O loop while executa um bloco de código enquanto uma condição específica for verdadeira.

Estrutura Básica do while

Descrição: Verifica a condição antes de executar o bloco de código.

var contador = 0

while contador < 5 {
    print(contador)
    contador += 1
}
// Saída:
// 0
// 1
// 2
// 3
// 4

Exemplo de Loop Infinito

Descrição: Tome cuidado para não criar loops infinitos acidentalmente.

var contador = 0

while true {
    print(contador)
    contador += 1
    if contador >= 5 {
        break
    }
}
// Saída:
// 0
// 1
// 2
// 3
// 4

Controle de Loop com continue

Descrição: Pula a iteração atual e continua com a próxima.

var contador = 0

while contador < 10 {
    contador += 1
    if contador % 2 == 0 {
        continue
    }
    print(contador)
}
// Saída:
// 1
// 3
// 5
// 7
// 9

Repeat-While Loop

O loop repeat-while é semelhante ao while, mas garante que o bloco de código seja executado pelo menos uma vez, pois a condição é verificada após a execução do bloco.

Estrutura Básica do repeat-while

Descrição: Executa o bloco de código uma vez antes de verificar a condição.

var contador = 0

repeat {
    print(contador)
    contador += 1
} while contador < 5
// Saída:
// 0
// 1
// 2
// 3
// 4

Comparação entre while e repeat-while

while Loop:

var contador = 0

while contador < 5 {
    print(contador)
    contador += 1
}
// Saída:
// 0
// 1
// 2
// 3
// 4

repeat-while Loop:

var contador = 0

repeat {
    print(contador)
    contador += 1
} while contador < 5
// Saída:
// 0
// 1
// 2
// 3
// 4

Boas Práticas ao Usar Loops

  • Evitar Loops Infinitos: Certifique-se de que a condição do loop eventualmente se tornará falsa. 
    var contador = 0

while true {
    print(contador)
    contador += 1
    if contador >= 5 {
        break  // Evita o loop infinito
    }
}
  • Usar for-in para Iterar sobre Coleções: Prefira for-in para iterar sobre arrays, dicionários, strings e outras coleções, pois é mais legível e seguro. 
    let frutas = ["Maçã", "Banana", "Laranja"]

for fruta in frutas {
    print(fruta)
}
  • Utilizar while para Condições Dinâmicas: Use while quando a quantidade de iterações não for conhecida antecipadamente e depender de uma condição dinâmica. 
    var contador = 0

while contador < 5 {
    print(contador)
    contador += 1
}
  • Garantir Execução com repeat-while: Use repeat-while quando precisar que o bloco de código seja executado pelo menos uma vez, independentemente da condição. 
    var contador = 0

repeat {
    print(contador)
    contador += 1
} while contador < 5
