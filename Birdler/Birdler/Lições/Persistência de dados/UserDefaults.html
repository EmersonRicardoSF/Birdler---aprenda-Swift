No desenvolvimento iOS, UserDefaults (ou User Defaults) é uma interface para interagir com as preferências do usuário no aplicativo. É comumente usado para armazenar pequenas quantidades de dados, como configurações ou preferências do usuário.

Como Funciona

UserDefaults armazena dados persistentemente entre as execuções do aplicativo. Aqui estão alguns pontos-chave sobre como funciona:

  • Chave-Valor: Os dados são armazenados usando um sistema de chave-valor.
  • Persistência: Os dados são armazenados localmente no dispositivo do usuário.
  • Acesso Simples: UserDefaults oferece métodos simples para salvar e recuperar dados.
  • Uso Comum: É frequentemente usado para salvar configurações do usuário, preferências ou estados de aplicativos simples.

Exemplo de Uso

Aqui está um exemplo simples de como usar UserDefaults em Swift para salvar e recuperar um valor:

import UIKit

// Salvando um valor
UserDefaults.standard.set("John Doe", forKey: "userName")

// Recuperando um valor
if let userName = UserDefaults.standard.string(forKey: "userName") {
    print("Usuário logado como: \(userName)")
} else {
    print("Nenhum usuário encontrado.")
}

Neste exemplo, "John Doe" é salvo com a chave "userName" nos UserDefaults. Quando o aplicativo é reiniciado, o valor associado à chave "userName" pode ser recuperado.

Métodos e Propriedades Principais

User Defaults oferece vários métodos e propriedades para gerenciar dados. Aqui estão alguns dos mais utilizados:

  • set(_:forKey:): Salva um valor para a chave especificada.
  • object(forKey:): Recupera um objeto para a chave especificada.
  • string(forKey:): Recupera uma string para a chave especificada.
  • integer(forKey:): Recupera um número inteiro para a chave especificada.
  • float(forKey:): Recupera um número de ponto flutuante para a chave especificada.
  • double(forKey:): Recupera um número de ponto flutuante de precisão dupla para a chave especificada.
  • bool(forKey:): Recupera um valor booleano para a chave especificada.
  • removeObject(forKey:): Remove o objeto associado à chave especificada.
  • synchronize(): Força a sincronização dos dados com disco (não mais necessário em versões recentes do iOS).

Considerações

UserDefaults é adequado para armazenar pequenas quantidades de dados não sensíveis, como preferências do usuário. No entanto, para dados mais complexos, sensíveis ou que requerem consultas sofisticadas, considere usar CoreData, armazenamento seguro ou outra solução adequada.
