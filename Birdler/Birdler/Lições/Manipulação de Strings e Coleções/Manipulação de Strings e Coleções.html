Manipulação de strings e coleções no contexto do UIKit, é um framework da Apple para construir interfaces gráficas no iOS. Vamos abordar alguns exemplos sobre manipulação de strings e coleções (arrays e dicionários) usando Swift, a linguagem de programação utilizada no desenvolvimento iOS com UIKit.

Manipulação de Strings em Swift

Concatenar Strings:


let string1 = "Olá"
let string2 = "Mundo"
let resultado = "\(string1) \(string2)"
print(resultado)  // Output: Olá Mundo

Substituir Substrings:


var texto = "Bom dia, Mundo!"
if let range = texto.range(of: "Mundo") {
    texto.replaceSubrange(range, with: "Swift")
}
print(texto)  // Output: Bom dia, Swift!

Dividir Strings:


let texto = "apple,banana,orange"
let frutas = texto.split(separator: ",")
print(frutas)  // Output: ["apple", "banana", "orange"]

Manipulação de Coleções em Swift

Arrays:

Inicializar e Acessar Elementos:


var frutas = ["maçã", "banana", "laranja"]
print(frutas[0])  // Output: maçã

Adicionar e Remover Elementos:


frutas.append("uva")
print(frutas)  // Output: ["maçã", "banana", "laranja", "uva"]

frutas.remove(at: 1)
print(frutas)  // Output: ["maçã", "laranja", "uva"]

Dicionários:

Inicializar e Acessar Valores:


var capitais = ["Brasil": "Brasília", "França": "Paris"]
print(capitais["Brasil"]!)  // Output: Brasília

Adicionar e Remover Pares Chave-Valor:


capitais["Itália"] = "Roma"
print(capitais)  // Output: ["Brasil": "Brasília", "França": "Paris", "Itália": "Roma"]

capitais.removeValue(forKey: "França")
print(capitais)  // Output: ["Brasil": "Brasília", "Itália": "Roma"]

Exemplo Prático com UIKit

Vamos combinar esses conceitos com um exemplo prático no UIKit. Suponha que temos um UILabel em nossa interface e queremos manipular o texto desse label.

Atualizando o Texto de um UILabel:


import UIKit

class ViewController: UIViewController {
    @IBOutlet weak var myLabel: UILabel!

    override func viewDidLoad() {
        super.viewDidLoad()
       
        // Inicializando o texto
        myLabel.text = "Olá Mundo!"
       
        // Manipulando o texto
        if let texto = myLabel.text {
            let novoTexto = texto.replacingOccurrences(of: "Mundo", with: "Swift")
            myLabel.text = novoTexto  // Output: Olá Swift!
        }
       
        // Exemplo com array
        let frutas = ["maçã", "banana", "laranja"]
        myLabel.text = frutas.joined(separator: ", ")  // Output: maçã, banana, laranja
    }
}
