No desenvolvimento iOS com UIKit, os métodos são funções que permitem interagir e manipular objetos da interface do usuário. Abaixo estão alguns métodos comuns:

Métodos de UIView

UIView é a classe base para todas as visualizações no UIKit.

  • addSubview(_:): Adiciona uma subview à hierarquia de visualização.
  • removeFromSuperview(): Remove a view de sua hierarquia de superview.
  • bringSubviewToFront(_:): Move a subview para a frente da hierarquia de visualização.
  • sendSubviewToBack(_:): Move a subview para trás da hierarquia de visualização.
  • isDescendant(of:): Verifica se a view é descendente de outra view.

Métodos de UIViewController

UIViewController gerencia uma hierarquia de visualização.

  • viewDidLoad(): Chamado após a carga de uma hierarquia de visualização.
  • viewWillAppear(_:): Chamado antes da exibição de uma view.
  • viewDidAppear(_:): Chamado após a exibição de uma view.
  • viewWillDisappear(_:): Chamado antes de uma view ser removida.
  • viewDidDisappear(_:): Chamado após a remoção de uma view.

Métodos de UIResponder

UIResponder é a classe base para responder a eventos de entrada.

  • touchesBegan(_:with:): Chamado quando toques começam em uma view.
  • touchesMoved(_:with:): Chamado quando toques movem em uma view.
  • touchesEnded(_:with:): Chamado quando toques terminam em uma view.
  • touchesCancelled(_:with:): Chamado quando toques são cancelados em uma view.

Métodos de UITableView

UITableView exibe dados em uma lista.

  • reloadData(): Recarrega todos os dados da tabela.
  • insertRows(at:with:): Insere novas linhas na tabela.
  • deleteRows(at:with:): Remove linhas da tabela.
  • scrollToRow(at:at:animated:): Faz a tabela rolar até uma linha específica.
