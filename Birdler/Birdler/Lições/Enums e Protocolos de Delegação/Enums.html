Os enums (ou enumerações) em Swift são um tipo poderoso que permite agrupar valores relacionados e trabalhar com eles de maneira segura e eficiente. No contexto do UIKit, os enums são amplamente utilizados para representar estados, tipos e opções que controlam o comportamento de componentes de interface do usuário. Vamos explorar como os enums são usados no UIKit, com exemplos detalhados para ilustrar suas aplicações.

Definindo e Usando Enums

1. Definição Básica de Enum

Um enum em Swift é definido usando a palavra-chave enum e pode ter valores associados chamados de "cases".


enum Estado {
    case ativo
    case inativo
    case suspenso
}

2. Enums com Valores Associados

Enums podem ter valores associados para cada case, permitindo armazenar informações adicionais.


enum Resultado {
    case sucesso(mensagem: String)
    case falha(erro: String)
}

Enums no Contexto do UIKit

1. Controlando Estados de Componentes

Um uso comum de enums no UIKit é controlar os estados de componentes de interface, como botões ou células de tabela.

Exemplo: Estado de um Botão


enum EstadoBotao {
    case normal
    case destacado
    case desativado
}

class CustomButton: UIButton {
    var estado: EstadoBotao = .normal {
        didSet {
            atualizarAparencia()
        }
    }

    private func atualizarAparencia() {
        switch estado {
        case .normal:
            self.backgroundColor = .systemBlue
        case .destacado:
            self.backgroundColor = .systemGreen
        case .desativado:
            self.backgroundColor = .systemGray
        }
    }
}

2. Tipos de Transições em UIViewController

O UIKit usa enums para definir os tipos de transições que podem ser usadas ao apresentar view controllers.


enum TransicaoViewController {
    case slide
    case fade
    case pop
}

func transicaoEntreViewControllers(tipo: TransicaoViewController) {
    let viewControllerAtual = self
    let novoViewController = UIViewController()

    switch tipo {
    case .slide:
        UIView.transition(from: viewControllerAtual.view, to: novoViewController.view, duration: 0.5, options: .transitionFlipFromRight, completion: nil)
    case .fade:
        UIView.transition(from: viewControllerAtual.view, to: novoViewController.view, duration: 0.5, options: .transitionCrossDissolve, completion: nil)
    case .pop:
        UIView.transition(from: viewControllerAtual.view, to: novoViewController.view, duration: 0.5, options: .transitionCurlUp, completion: nil)
    }
}

3. Controlando o Layout de Componentes

Enums também são usados para controlar o layout de componentes no UIKit, como a direção do empilhamento em uma UIStackView.


enum DirecaoEmpilhamento {
    case horizontal
    case vertical
}

class CustomStackView: UIStackView {
    var direcao: DirecaoEmpilhamento = .vertical {
        didSet {
            self.axis = direcao == .vertical ? .vertical : .horizontal
        }
    }
}

Enums Utilizados no UIKit

O UIKit define muitos enums integrados que são usados para configurar e controlar diversos aspectos da interface do usuário.

1. UIControl.State

Este enum define os diferentes estados de um controle, como um botão.


public struct UIControl.State : OptionSet {
    public let rawValue: Int
    public static let normal: UIControl.State
    public static let highlighted: UIControl.State
    public static let disabled: UIControl.State
    public static let selected: UIControl.State
    public static let focused: UIControl.State
    public static let application: UIControl.State
    public static let reserved: UIControl.State
}

Exemplo de Uso


let botao = UIButton(type: .system)
botao.setTitle("Normal", for: .normal)
botao.setTitle("Destacado", for: .highlighted)
botao.setTitle("Desativado", for: .disabled)

2. UIView.ContentMode

Este enum define como o conteúdo de uma view deve ser ajustado quando o layout da view muda.


enum ContentMode: Int {
    case scaleToFill
    case scaleAspectFit
    case scaleAspectFill
    case redraw
    case center
    case top
    case bottom
    case left
    case right
    case topLeft
    case topRight
    case bottomLeft
    case bottomRight
}

Exemplo de Uso


let imageView = UIImageView()
imageView.contentMode = .scaleAspectFit

3. UIActivityIndicatorView.Style

Este enum define os estilos de um UIActivityIndicatorView.


enum Style : Int {
    case medium
    case large
}

Exemplo de Uso


let activityIndicator = UIActivityIndicatorView(style: .large)
activityIndicator.startAnimating()

Conclusão

Os enums são uma funcionalidade poderosa e flexível do Swift que permitem definir conjuntos de valores relacionados e trabalhar com eles de maneira segura e eficiente. No contexto do UIKit, os enums são amplamente utilizados para controlar estados, tipos e opções de componentes de interface do usuário, tornando o código mais legível, seguro e fácil de manter.
