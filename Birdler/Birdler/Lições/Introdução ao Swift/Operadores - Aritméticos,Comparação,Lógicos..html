Operadores Aritméticos

Os operadores aritméticos são usados para realizar operações matemáticas básicas. Vamos explorar cada um deles em detalhes.

Adição (+)

Descrição: Soma dois números.

let a = 5
let b = 3
let soma = a + b  // Resultado: 8

Subtração (-)

Descrição: Subtrai o segundo número do primeiro.

let a = 10
let b = 2
let subtracao = a - b  // Resultado: 8

Multiplicação (*)

Descrição: Multiplica dois números.

let a = 4
let b = 2
let multiplicacao = a * b  // Resultado: 8

Divisão (/)

Descrição: Divide o primeiro número pelo segundo. Em divisões inteiras, o resultado é truncado.

let a = 20
let b = 4
let divisao = a / b  // Resultado: 5

Divisão com Double:

let x: Double = 20.0
let y: Double = 4.0
let divisaoDouble = x / y  // Resultado: 5.0

Módulo (%)

Descrição: Retorna o resto da divisão do primeiro número pelo segundo.

let a = 7
let b = 3
let modulo = a % b  // Resultado: 1

Exemplos Combinados de Operadores Aritméticos

let x = 15
let y = 4

let soma = x + y  // 19
let diferenca = x - y  // 11
let produto = x * y  // 60
let quociente = x / y  // 3 (divisão inteira)
let resto = x % y  // 3

Operadores de Comparação

Os operadores de comparação são usados para comparar dois valores e retornar um valor booleano (true ou false). Vamos detalhar cada um deles.

Igual a (==)

Descrição: Verifica se dois valores são iguais.

let a = 5
let b = 5
let isEqual = (a == b)  // true

Diferente de (!=)

Descrição: Verifica se dois valores são diferentes.

let a = 5
let b = 3
let isNotEqual = (a != b)  // true

Maior que (>)

Descrição: Verifica se o primeiro valor é maior que o segundo.

let a = 7
let b = 5
let isGreater = (a > b)  // true

Menor que (<)

Descrição: Verifica se o primeiro valor é menor que o segundo.

let a = 3
let b = 5
let isLesser = (a < b)  // true

Maior ou igual a (>=)

Descrição: Verifica se o primeiro valor é maior ou igual ao segundo.

let a = 6
let b = 6
let isGreaterOrEqual = (a >= b)  // true

Menor ou igual a (<=)

Descrição: Verifica se o primeiro valor é menor ou igual ao segundo.

let a = 4
let b = 5
let isLesserOrEqual = (a <= b)  // true

Exemplos Combinados de Operadores de Comparação

let x = 10
let y = 20

let igual = (x == y)  // false
let diferente = (x != y)  // true
let maior = (x > y)  // false
let menor = (x < y)  // true
let maiorOuIgual = (x >= y)  // false
let menorOuIgual = (x <= y)  // true

Operadores Lógicos

Os operadores lógicos são usados para combinar múltiplas condições booleanas. Vamos explorar cada um deles.

AND lógico (&&)

Descrição: Retorna true se ambas as condições forem verdadeiras.

let a = true
let b = false
let andResult = (a && b)  // false

OR lógico (||)

Descrição: Retorna true se pelo menos uma das condições for verdadeira.

let a = true
let b = false
let orResult = (a || b)  // true

NOT lógico (!)

Descrição: Inverte o valor booleano.

let a = true
let notResult = !a  // false

Exemplos Combinados de Operadores Lógicos

let idade = 22
let temCarteiraDeMotorista = true

let podeDirigir = (idade >= 18) && temCarteiraDeMotorista  // true

if podeDirigir {
    print("Pode dirigir")
} else {
    print("Não pode dirigir")
}

let estaChovendo = false
let temGuardaChuva = true

let vaiSair = (estaChovendo && temGuardaChuva) || !estaChovendo  // true

if vaiSair {
    print("Vai sair de casa")
} else {
    print("Não vai sair de casa")
}
