O que são Constantes?

Constantes são espaços de memória reservados para armazenar dados que, uma vez atribuídos, não podem ser alterados durante a execução do programa. Elas são úteis para valores que você sabe que não mudarão, como pi, nomes de usuário, ou configurações de aplicação que permanecem constantes.

Declaração de Constantes

Para declarar uma constante, usamos a palavra-chave let, seguida pelo nome da constante, seu tipo e um valor inicial.

let pi: Double = 3.14159
let anoDeNascimento: Int = 1995


    
Modificando o valor de uma Constante

    
Diferente das variáveis, o valor de uma constante não pode ser alterado após ser definido. Tentar fazer isso resultará em um erro de compilação.

    
let pi = 3.14159
pi = 3.14  // Erro: Cannot assign to value: 'pi' is a 'let' constant


    
Vantagens de Usar Constantes

    
    
        
  • Segurança: Previne a modificação acidental de valores que não deveriam mudar.
    • 
        
  • Clareza: Deixa claro para outros desenvolvedores que o valor não deve ser alterado.
    • 
        
  • Otimização: O compilador pode otimizar o código sabendo que certos valores são constantes.
    • 
    


    
Boas Práticas

    
    
        
  • Use let sempre que possível: Declare uma constante se você não precisa que o valor mude, o que ajuda a evitar bugs.
    • 
        
  • Nomeie claramente: Dê nomes significativos às constantes para facilitar a leitura do código.
