Swift é o que é conhecido como uma linguagem segura quanto a tipos, o que significa que cada variável deve ser de um tipo específico. A variável str que o Xcode criou para nós contém uma sequência de letras que formam “Hello, playground”, então Swift atribui a ela o tipo String.

Por outro lado, se quisermos armazenar a idade de alguém, podemos criar uma variável assim:



var age = 38

Isso contém um número inteiro, então Swift atribui o tipo Int – abreviação de “integer” (inteiro).

Se você tiver números grandes, Swift permite que você use sublinhados como separadores de milhares – eles não mudam o número, mas tornam mais fácil a leitura. Por exemplo:



var population = 8_000_000

Strings e inteiros são tipos diferentes e não podem ser misturados. Então, enquanto é seguro mudar str para “Goodbye”, eu não posso torná-la 38 porque isso é um Int e não um String.
