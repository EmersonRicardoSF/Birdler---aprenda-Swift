Utilizando Variáveis em Swift

Introdução

Em Swift, variáveis são utilizadas para armazenar valores que podem ser modificados ao longo do tempo. Elas são declaradas usando a palavra-chave var.

Declaração de Variáveis

Para declarar uma variável, utilize a sintaxe:

var nomeVariavel: Tipo = valorInicial

Exemplo:

var nome: String = "John"
var idade: Int = 25

Inferência de Tipo

Swift pode inferir o tipo da variável a partir do valor inicial, permitindo declarações mais concisas:

var cidade = "São Paulo" // Inferido como String
var pontuacao = 95       // Inferido como Int

Modificação de Variáveis

O valor de uma variável pode ser modificado após a declaração:

var contador = 10
contador = 20

Constantes

Para valores que não devem ser alterados, use let em vez de var:

let pi = 3.14159

Conclusão

Variáveis em Swift são essenciais para armazenar e manipular dados. A flexibilidade da linguagem permite tanto a declaração explícita quanto a inferência de tipos, facilitando o desenvolvimento de código eficiente e claro.
