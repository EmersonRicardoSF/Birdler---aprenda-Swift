Sintaxe Básica: Variáveis, Constantes e Tipos de Dados

Nesta seção, vamos aprofundar no uso de variáveis, constantes e tipos de dados em Swift. Entender como declarar e usar esses elementos é fundamental para a programação em Swift. Vamos explorar cada um desses conceitos em detalhe.

Variáveis

Variáveis são usadas para armazenar valores que podem ser modificados ao longo do tempo. Para declarar uma variável em Swift, utilizamos a palavra-chave var e o nome da variável, como por exemplo: nome. Se a variável for formada por uma frase, sempre usamos a primeira letra em minúsculo e as outras palavras começamos em maiúsculo para melhor identificar a frase, e isso será para todos os valores que declaramos na programação.

Declaração e Inicialização

var nome: String = "João"

Aqui, estamos declarando uma variável chamada nome do tipo String e inicializando-a com o valor "João". A sintaxe é var nomeVariavel: Tipo = valor.

Modificação de Valores

var idade: Int = 25
idade = 26

Neste exemplo, declaramos a variável idade do tipo Int com o valor 25. Posteriormente, modificamos o valor para 26.

Inferência de Tipo

Swift é uma linguagem com tipagem estática, mas também é capaz de inferir o tipo de uma variável com base no valor atribuído a ela, permitindo que você omita a anotação de tipo. 

var cidade = "São Paulo" // Swift infere que `cidade` é do tipo String
