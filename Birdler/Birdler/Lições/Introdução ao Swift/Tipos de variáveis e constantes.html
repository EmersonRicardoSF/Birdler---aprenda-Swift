String

O que é String?

  • String é um tipo de dado que representa uma sequência de caracteres, como texto.
  • Strings em Swift são Unicode, o que significa que elas podem representar quase todos os caracteres modernos e antigos usados na codificação de texto.

Declaração e Inicialização

Você pode declarar e inicializar uma variável ou constante do tipo String usando a palavra-chave var ou let.

let saudacao: String = "Olá, Mundo!"
var nome: String = "Maria"

Concatenação de Strings

Concatenação é o processo de juntar duas ou mais strings em uma única string. Em Swift, isso é feito usando o operador de adição +.

let primeiroNome = "João"
let ultimoNome = "Silva"
let nomeCompleto = primeiroNome + " " + ultimoNome  // "João Silva"

Aqui, " " (um espaço) é usado para separar o primeiro nome do último nome na string concatenada.

Interpolação de Strings

Interpolação de strings é uma forma conveniente de inserir valores de variáveis dentro de strings literais. Em Swift, isso é feito usando a barra invertida \ seguida por parênteses () contendo a expressão ou a variável a ser interpolada.

let idade = 25
let mensagem = "Eu tenho \(idade) anos."  // "Eu tenho 25 anos."

Você também pode usar expressões dentro da interpolação:

let comprimento = 10
let largura = 5
let area = comprimento * largura
let descricaoArea = "A área do retângulo é \(area) metros quadrados."  // "A área do retângulo é 50 metros quadrados."

Manipulação de Strings

Manipulação de strings envolve diversas operações que podemos realizar com strings, como alterar o caso das letras, contar caracteres, dividir strings, substituir substrings, entre outras.

  • Alterar o Caso das Letras: 
    let frase = "Swift é incrível!"
let fraseMaiuscula = frase.uppercased()  // "SWIFT É INCRÍVEL!"
let fraseMinuscula = frase.lowercased()  // "swift é incrível!"
  • Contar Caracteres: 
    let quantidadeDeCaracteres = frase.count  // 17
  • Verificar Prefixo e Sufixo: 
    let mensagem = "Olá, Mundo!"
let temPrefixo = mensagem.hasPrefix("Olá")  // true
let temSufixo = mensagem.hasSuffix("Mundo!")  // true
  • Dividir String: 
    let frutas = "Maçã, Banana, Laranja"
let listaDeFrutas = frutas.split(separator: ", ")  // ["Maçã", "Banana", "Laranja"]
  • Substituir Substring: 
    let saudacao = "Olá, Mundo!"
let novaSaudacao = saudacao.replacingOccurrences(of: "Mundo", with: "Swift")  // "Olá, Swift!"
  • Acessar Caracteres Específicos: 
    let texto = "Swift"
let primeiroCaracter = texto[texto.startIndex]  // "S"
let ultimoCaracter = texto[texto.index(before: texto.endIndex)]  // "t"
  • Inserir e Remover Substrings: 
    var mensagem = "Olá, Mundo!"
mensagem.insert("s", at: mensagem.index(before: mensagem.endIndex))  // "Olá, Mundos!"
mensagem.remove(at: mensagem.index(before: mensagem.endIndex))  // "Olá, Mundo!"

Int

O que é Int?

  • Int é um tipo de dado que representa números inteiros, ou seja, números sem parte fracionária.
  • Em Swift, o tipo padrão de inteiro é Int, que é otimizado para a arquitetura da plataforma (32 bits ou 64 bits).

Declaração e Inicialização

Você pode declarar uma variável ou constante do tipo Int usando a palavra-chave var ou let.

let idade: Int = 30
var quantidadeDeItens: Int = 50

Swift é capaz de inferir o tipo Int quando não é explicitamente declarado, desde que o valor inicial seja um inteiro.

let numero = 100  // Inferido como Int

Double (Números de Ponto Flutuante de Precisão Dupla)

O que é?

  • Double é um tipo de dado que representa números com ponto flutuante de precisão dupla, o que significa que pode armazenar números com maior precisão e uma faixa maior de valores do que Float.
  • Usado quando a precisão dos valores decimais é crucial.

Declaração e Inicialização

Você pode declarar uma variável ou constante do tipo Double usando a palavra-chave var ou let.

let altura: Double = 1.75
var peso: Double = 70.5

Swift é capaz de inferir o tipo Double quando não é explicitamente declarado, desde que o valor inicial seja um número decimal.

let temperatura = 23.5  // Inferido como Double

Float (Números de Ponto Flutuante de Precisão Simples)

O que é?

  • Float é um tipo de dado que representa números com ponto flutuante de precisão simples.
  • Usado quando a precisão dos valores decimais não é tão crítica e se deseja economizar memória.

Declaração e Inicialização

Você pode declarar uma variável ou constante do tipo Float usando a palavra-chave var ou let.

let distancia: Float = 5.6
var velocidade: Float = 3.4

Swift é capaz de inferir o tipo Float quando o valor inicial é explicitamente um Float.

let preco: Float = 19.99

Bool (Booleanos)

O que é?

  • Bool é um tipo de dado que representa valores booleanos, ou seja, valores que só podem ser true ou false.
  • Usado para representar condições lógicas e controlar o fluxo do programa com instruções condicionais.

Declaração e Inicialização

Você pode declarar uma variável ou constante do tipo Bool usando a palavra-chave var ou let.

let isSwiftFun: Bool = true
var isRaining: Bool = false

Swift é capaz de inferir o tipo Bool quando o valor inicial é explicitamente true ou false.
