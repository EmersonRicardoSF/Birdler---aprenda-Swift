O que é uma API RESTful?

Uma API RESTful é uma maneira padronizada de acessar dados e funcionalidades em um servidor usando a internet. Pense nisso como um menu em um restaurante, onde você faz pedidos específicos e recebe respostas de acordo. Cada pedido que você faz é uma solicitação HTTP (GET, POST, PUT ou DELETE).

Tipos de Solicitações

  • GET: Pede dados do servidor. É como perguntar "Qual é o clima hoje?".
  • POST: Envia novos dados para o servidor. É como enviar uma nova mensagem.
  • PUT: Atualiza dados existentes no servidor. É como corrigir um erro em uma mensagem que você já enviou.
  • DELETE: Remove dados do servidor. É como pedir para apagar uma mensagem enviada.

Como Consumir APIs RESTful em Swift

Vamos ver exemplos de como fazer cada tipo de solicitação em Swift usando URLSession.

1. GET

Para buscar dados do servidor, como informações de clima:


import Foundation

if let url = URL(string: "https://api.clima.com/hoje") {
    let task = URLSession.shared.dataTask(with: url) { data, response, error in
        if let data = data {
            let dataString = String(data: data, encoding: .utf8)
            print("Dados recebidos: \(dataString ?? "Nenhum dado")")
        }
    }
    task.resume()
}

2. POST

Para enviar novos dados ao servidor, como criar uma nova tarefa:


import Foundation

if let url = URL(string: "https://api.tarefas.com/nova") {
    var request = URLRequest(url: url)
    request.httpMethod = "POST"
    request.setValue("application/json", forHTTPHeaderField: "Content-Type")
    let newTask = ["title": "Estudar Swift", "completed": false] as [String : Any]
    request.httpBody = try? JSONSerialization.data(withJSONObject: newTask)
    
    let task = URLSession.shared.dataTask(with: request) { data, response, error in
        if let data = data {
            let dataString = String(data: data, encoding: .utf8)
            print("Resposta: \(dataString ?? "Nenhuma resposta")")
        }
    }
    task.resume()
}

3. PUT

Para atualizar dados existentes no servidor, como corrigir uma tarefa:


import Foundation

if let url = URL(string: "https://api.tarefas.com/atualizar/1") {
    var request = URLRequest(url: url)
    request.httpMethod = "PUT"
    request.setValue("application/json", forHTTPHeaderField: "Content-Type")
    let updatedTask = ["title": "Estudar Swift Avançado", "completed": true] as [String : Any]
    request.httpBody = try? JSONSerialization.data(withJSONObject: updatedTask)
    
    let task = URLSession.shared.dataTask(with: request) { data, response, error in
        if let data = data {
            let dataString = String(data: data, encoding: .utf8)
            print("Resposta: \(dataString ?? "Nenhuma resposta")")
        }
    }
    task.resume()
}

4. DELETE

Para remover dados do servidor, como deletar uma tarefa:


import Foundation

if let url = URL(string: "https://api.tarefas.com/deletar/1") {
    var request = URLRequest(url: url)
    request.httpMethod = "DELETE"
    
    let task = URLSession.shared.dataTask(with: request) { data, response, error in
        if let data = data {
            let dataString = String(data: data, encoding: .utf8)
            print("Resposta: \(dataString ?? "Nenhuma resposta")")
        }
    }
    task.resume()
}

Conclusão

Consumir APIs RESTful em Swift é essencial para fazer aplicativos que se comunicam com a internet. Com as solicitações GET, POST, PUT e DELETE, você pode buscar, criar, atualizar e deletar dados em servidores, integrando funcionalidades online de forma eficiente no seu aplicativo.
