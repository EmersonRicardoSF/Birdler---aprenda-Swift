As extensões em Swift permitem adicionar novas funcionalidades a uma classe, estrutura, enumeração ou protocolo existente. No contexto do UIKit, podemos usar extensões para adicionar métodos de conveniência, personalizar a aparência e comportamento de componentes da interface do usuário e organizar melhor o código.

Exemplos de Extensões no UIKit

1. Extensão para UIColor

Adicionando cores personalizadas ao UIColor para reutilização fácil.


import UIKit

extension UIColor {
    static let customRed = UIColor(red: 0.8, green: 0.0, blue: 0.0, alpha: 1.0)
    static let customGreen = UIColor(red: 0.0, green: 0.8, blue: 0.0, alpha: 1.0)
    static let customBlue = UIColor(red: 0.0, green: 0.0, blue: 0.8, alpha: 1.0)
}

2. Extensão para UIView

Adicionando métodos de conveniência para manipulação de layout.


import UIKit

extension UIView {
    func setCornerRadius(_ radius: CGFloat) {
        self.layer.cornerRadius = radius
        self.layer.masksToBounds = true
    }

    func setBorderColor(_ color: UIColor, width: CGFloat) {
        self.layer.borderColor = color.cgColor
        self.layer.borderWidth = width
    }
}

3. Extensão para UIImage

Adicionando funcionalidades para redimensionamento e arredondamento de imagens.


import UIKit

extension UIImage {
    func resized(to size: CGSize) -> UIImage? {
        UIGraphicsBeginImageContextWithOptions(size, false, 0.0)
        draw(in: CGRect(origin: .zero, size: size))
        let resizedImage = UIGraphicsGetImageFromCurrentImageContext()
        UIGraphicsEndImageContext()
        return resizedImage
    }

    func roundedImage() -> UIImage? {
        let minEdge = min(size.width, size.height)
        let squareSize = CGSize(width: minEdge, height: minEdge)
        let origin = CGPoint(x: (size.width - minEdge) / 2, y: (size.height - minEdge) / 2)

        UIGraphicsBeginImageContextWithOptions(squareSize, false, 0.0)
        UIBezierPath(roundedRect: CGRect(origin: .zero, size: squareSize), cornerRadius: minEdge / 2).addClip()
        draw(at: CGPoint(x: -origin.x, y: -origin.y))
        let roundedImage = UIGraphicsGetImageFromCurrentImageContext()
        UIGraphicsEndImageContext()

        return roundedImage
    }
}

4. Extensão para UIViewController

Adicionando métodos de conveniência para apresentar alertas.


import UIKit

extension UIViewController {
    func showAlert(title: String, message: String) {
        let alertController = UIAlertController(title: title, message: message, preferredStyle: .alert)
        let okAction = UIAlertAction(title: "OK", style: .default, handler: nil)
        alertController.addAction(okAction)
        present(alertController, animated: true, completion: nil)
    }
}

Conclusão

As extensões são uma maneira poderosa de aumentar a funcionalidade das classes do UIKit sem a necessidade de subclassificação. Elas ajudam a manter o código modular, reutilizável e organizado. É importante usá-las de maneira sensata para evitar sobrecarregar uma única classe com muitas responsabilidades.
